# Legal Rights Protection Forum (LRPF) Website

This repository contains the source code for the official website of the **Legal Rights Protection Forum (LRPF)**, a non-profit organization dedicated to upholding India's constitutional framework, national security, and legal integrity.

## About LRPF

The Legal Rights Protection Forum (LRPF) is a team of passionate legal and social activists, lawyers, and researchers who operate on the frontlines of legal activism. 

### Our Mission

Our mission is to build a lawful, transparent, and secure India by holding organizations and individuals accountable for violations of national law. We believe in empowering citizens with the knowledge and tools they need to protect their rights and contribute to a more just and secure society.

## Website Features

The website provides information about the organization, its work, and how to get involved. The key sections are:

*   **Home:** Main landing page with a brief introduction to the organization.
*   **About:** Detailed information about the organization's mission and work.
*   **Our Work:** Describes the key areas of focus for the organization.
*   **Get Involved:** Information on how to contribute to the organization.
*   **Resources:** A knowledge bank with legal resources.
*   **Gallery:** A photo gallery of the organization's activities.
*   **Contact:** Contact information and a form to get in touch.
*   **Report Grievance:** A form to report a grievance.

## How to Run the Website

This is a static website built with HTML, CSS, and JavaScript. No special build process is required.

To view the website, you can simply open the `index.html` file in your web browser.

Alternatively, you can use a simple HTTP server to serve the files. For example, if you have Python installed, you can run the following command in the root directory of the project:

```bash
python -m http.server
```

This will start a web server on port 8000. You can then view the website by navigating to `http://localhost:8000` in your web browser.
